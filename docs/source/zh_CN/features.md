# 功能展示

此页面展示了本文档站点中可用的所有功能。

## 代码块

### Python 代码

```python
def hello_world():
    """一个简单的 hello world 函数。"""
    print("你好，世界！")
    return True

if __name__ == "__main__":
    result = hello_world()
    print(f"结果: {result}")
```

### JavaScript 代码

```javascript
function greet(name) {
    console.log(`你好，${name}！`);
    return true;
}

// 调用函数
greet("世界");
```

### Bash 命令

```bash
# 安装依赖
pip install sphinx myst-parser

# 构建文档
make html

# 本地服务
python -m http.server 8000
```
